# Vietnam Food Check 🍲

Website giới thiệu ẩm thực Việt Nam với Chatbox AI

## Tính Năng
- Giới thiệu 100+ món ăn Việt Nam
- Phân loại theo khu vực (Bắc-Trung-Nam)
- Tìm kiếm và bộ lọc hiện đại
- Chatbox AI Gemini tư vấn
- Giao diện đẹp, responsive

## Công Nghệ
- Frontend: HTML5, CSS3, JavaScript
- Backend: Node.js + Express
- Database: MySQL
- AI: Gemini API

## Cài Đặt

### Prerequisites
- Node.js v16+
- MySQL Server
- Gemini API Key

### Setup

1. Clone repository
```bash
git clone https://github.com/hizako123/vietnamfoodcheck.git
cd vietnamfoodcheck
```

2. Install dependencies
```bash
npm install
```

3. Configure .env
```bash
cp .env.example .env
# Edit .env với MySQL credentials của bạn
```

4. Start server
```bash
npm start
```
